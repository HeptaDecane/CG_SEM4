# CGL_MiniProject

Execution Commands in Ubuntu:
    g++ main.cpp -lglut -lGL -lGLEW -lGLU -o main.exe
    ./main.exe

Requirements:
    freeglut3
    freeglut3-dev
    binutils-gold
    g++
    cmake
    libglew-dev
    mesa-common-dev
    build-essential
    libglew1.5-dev
    libglm-dev

Installation and Setup: 
    https://www.wikihow.com/Install-Mesa-(OpenGL)-on-Linux-Mint
